# 🎵 Aurea — Premium Music Player

> A premium music player with a luxurious **Golden & Cream** theme. Built with **HTML, CSS & JavaScript** — no frameworks, no libraries!

---

## ✨ Features

- 🎵 **100+ Songs** library with covers & artists  
- 🔍 **Real-time Search** with "Not Found" handling  
- 🧭 **Explore** by Genre — Bollywood, Punjabi, Pop, Rock, Classical, Retro  
- ❤️ **Liked Songs** — Save favorites (localStorage)  
- 🎧 **Fully Functional Player** — Play, Pause, Next, Prev, Volume, Progress  
- 📱 **100% Responsive** — Mobile, Tablet, Laptop, Desktop  
- 🎨 **Premium Golden + Cream Theme**  
- ⚡ **Smooth Animations**  
- ⌨️ **Keyboard Shortcuts** — Spacebar for Play/Pause
