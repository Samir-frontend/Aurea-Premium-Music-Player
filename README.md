## 🎵 Overview

> **Aurea Premium** is a beautifully crafted, feature-rich music player web app with a sleek dark UI, smooth animations, and an intuitive layout — built entirely with vanilla HTML, CSS & JavaScript. Inspired by premium music platforms like Spotify, Aurea delivers a refined listening experience right in the browser.

---

## ✨ Features

### 🎧 Core Player
- ▶️ **Play / Pause** — Instant toggle with smooth icon transitions
- ⏭️ **Next / Previous Track** — Navigate through your playlist seamlessly
- 🔀 **Shuffle Mode** — Randomize your listening order
- 🔁 **Repeat Mode** — Loop a single track or the entire playlist
- 🎚️ **Volume Control** — Smooth slider with mute toggle
- ⏱️ **Progress Bar** — Seek to any point in the track with a click

### 📚 Library & Navigation
- 🗂️ **Sidebar Navigation** — Clean slide-in sidebar with section links
- 📖 **Music Library** — Browse all available tracks in one place
- 🔍 **Search** — Instantly find songs from your collection
- ❤️ **Liked Songs** — Save your favorite tracks with one click
- ⭐ **Favorites** — Curate a personal favorites playlist

### 🎨 UI / UX
- 🌑 **Premium Dark Theme** — Gold & dark color palette
- 💫 **Smooth Animations** — CSS transitions on every interaction
- 📱 **Fully Responsive** — Works perfectly on mobile, tablet & desktop
- 🖼️ **Album Art Display** — Dynamic cover art per track
- 🎵 **Now Playing Indicator** — Visual highlight on active track
- ✨ **Glassmorphism Effects** — Modern frosted-glass UI elements

## ⚙️ How It Works

```javascript
1. Sidebar renders navigation — Library, Search, Liked, Favorites
2. Tracks load dynamically into the player queue
3. Play/Pause toggles audio state & animates the UI
4. Progress bar syncs with audio currentTime in real-time
5. Shuffle picks a random unplayed track each time
6. Liked & Favorites stored for the session
7. Search filters tracks live as you type
```
